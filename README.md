# OmniRing

An app to monitor and collect sensor data from Omni Ring, with firmware upgrade capability.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

3. Prebuild the app
    ```bash
    npx expo prebuild --clean
    ```

## Run on iOS

When you're ready, navigate to `/ios`, open `omniRing.xcodeproj`, select a real iPhone (simulator won't work for bluetooth), then hit Play.

Configure development team as necessary.
